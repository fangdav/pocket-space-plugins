# Pocket Space Plugins

Plugin marketplace for the Pocket Space shared drive.

## Installation

```shell
/plugin marketplace add your-org/pocket-space-plugins
/plugin install pocket-space@pocket-space-plugins
```

## Plugins

### pocket-space

Guidelines and research workflow for contributing to the Pocket Space shared drive.

**Foundation skills:**
- **contribution-guidelines** — File formats, naming conventions, author sign-off, README standards, research folder rule
- **folder-navigation** — Folder map and routing guide

**Research workflow:**
1. **research-md-maker** — Broad landscape research with 6-12 diverse segments
2. **analyzer** — Comparison matrix ranking segments across criteria
3. **deep-researcher** — Detailed deep dives on top-ranked options
4. **research-synthesizer** — Unified .docx deliverable from all research files
5. **comment-reviewer** — Process teammate feedback into actionable editing guide
